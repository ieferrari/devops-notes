## Linux Installation
from [official docs](https://redis.io/topics/quickstart):


    wget http://download.redis.io/redis-stable.tar.gz
    tar xvzf redis-stable.tar.gz
    cd redis-stable
    make

    sudo make install

to start redis

    redis-server

in other terminal opend command-line for redis

    redis-cli

check redis performance

    redis-benchmark
